# Autowartung

## Beschreibung
Erstelle eine app zum verwalten von Auto Wartungen (z.B. Ölwechsel)
mit PWA, Firebase, offline support, real-time   sync, and PWA install capability
zuerst aber erst mal alles lokal testen und erst später onlinestellen
Es sollen mehrere Fahrzeuge erfasst werden können mit Fahrzeugtyp, Kennzeichen, Fahrzeugnutzer, aktueller Kilometerstand
Bei den Wartungen starten wir mal mit:
Ölwechsel, Bremsen vorne, Bremsen hinten, Starterbatterie
soll aber erweiterbar sein
jede Wartung soll mit Datum und Kilometerstand erfasst werden
es soll für jeder Wartung eine Erneuerung geplant werden
Beispiel Ölwechsel muss nach 30.000km oder 24 Monaten erneuert werden
dazu am besten eine Erinnerung per WhatsApp versenden (aber zuerst mal ohne das starten)
es sollen passende Icons für die Wartungen angezeigt werden
es soll auch möglich sein den Kilometerstand bei bedarf mit Datum zu aktualisieren
